# Artha — Human Evaluation Scoring Rubric

## Instructions for Annotators

You are evaluating responses from an AI personal finance assistant.
For each response, score it on four dimensions (1-5).
Score independently — do not discuss scores with other annotators.

Each response was generated by the AI in answer to a user's question
about their personal finances (spending, budgets, goals, subscriptions, etc.)

---

## Scoring Dimensions

### 1. Data Accuracy (1-5)

Does the response cite **correct, specific numbers** from the user's
financial data?

| Score | Meaning |
|-------|---------|
| 5 | Multiple correct dollar amounts, merchant names, dates cited |
| 4 | Most numbers correct; minor imprecision (e.g., rounding) |
| 3 | Some correct data cited, but missing key figures |
| 2 | Vague or partially wrong numbers |
| 1 | No specific data cited, or numbers are clearly wrong |

### 2. Insight Quality (1-5)

Is the financial reasoning **sound and specific** to this user's
situation?

| Score | Meaning |
|-------|---------|
| 5 | Identifies patterns specific to this user's behavior (e.g., "your fast-food spending increased 30% vs. last month") |
| 4 | Good analysis with some user-specific observations |
| 3 | Reasonable analysis but somewhat generic |
| 2 | Mostly generic advice ("reduce spending", "save more") |
| 1 | Irrelevant or incorrect financial reasoning |

### 3. Actionability (1-5)

Does the response provide **concrete, prioritized next steps**?

| Score | Meaning |
|-------|---------|
| 5 | Specific actions with dollar amounts (e.g., "cancel Netflix at $15.99/mo to save $192/yr") |
| 4 | Clear suggestions with some specifics |
| 3 | Reasonable suggestions but lacks prioritization or specifics |
| 2 | Vague suggestions ("try to save more") |
| 1 | No actionable suggestions |

### 4. Completeness (1-5)

Does the response surface **all critical financial signals** present
in the data?

| Score | Meaning |
|-------|---------|
| 5 | Catches all major patterns (overdraft fees, unusual charges, high spending categories, subscription changes) |
| 4 | Catches most important signals; misses 1 minor item |
| 3 | Catches the main point but misses 2-3 secondary signals |
| 2 | Misses several important signals |
| 1 | Superficial response that misses most financial signals |

---

## Pass/Fail Threshold

A response **passes** if it scores ≥ 3 on ALL FOUR dimensions.
Any dimension scoring 1 or 2 means the response fails.
